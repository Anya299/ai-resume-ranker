# 🚀 AI-Powered Resume Ranking & Screening System

An intelligent **AI-driven Applicant Tracking System (ATS)** that automatically ranks resumes based on job descriptions using Natural Language Processing (NLP). It helps simulate real-world hiring systems by analyzing skill matches, ranking candidates, and identifying skill gaps.

---

## 🧠 Project Overview

This project automates resume screening using Machine Learning techniques. It compares multiple resumes against a job description and ranks candidates based on semantic similarity and skill relevance.

It also highlights:
- Key skills required for the job
- Skills present in each resume
- Missing skills for improvement

---

## 🎯 Key Features

✔ AI-based Resume Ranking System  
✔ NLP-powered Skill Matching (TF-IDF + Cosine Similarity)  
✔ Missing Skill Detection per candidate  
✔ Resume Parsing from PDF files  
✔ Interactive Web App using Streamlit  
✔ Real-time candidate evaluation  

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Machine Learning:** Scikit-learn  
- **NLP Techniques:** TF-IDF Vectorization, Cosine Similarity  
- **Data Processing:** Pandas, NumPy  
- **Frontend:** Streamlit  
- **File Handling:** PDFPlumber  

---

## 🧩 How It Works

1. Job description is provided by the user  
2. Multiple resumes are uploaded (PDF format)  
3. Text is extracted and cleaned  
4. TF-IDF converts text into numerical vectors  
5. Cosine similarity calculates match scores  
6. System ranks resumes based on similarity  
7. Skill gaps are identified for each candidate  

---

## 📊 Output Features

- Ranked list of candidates  
- Match percentage score  
- Top job-related skills extracted  
- Missing skills per resume
